# 104. Maximum Depth of Binary Tree

**Difficulty:** Easy | **Topic:** Binary Tree, DFS | **Date:** 2026-04-01

---

## 题目 / Problem

给定一个二叉树，找出其最大深度。
最大深度是从根节点到最远叶子节点的最长路径上的节点数。

Given a binary tree, find its maximum depth.
The maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.

---

## 思路 / Approach

**递归（DFS）/ Recursive DFS**

二叉树的最大深度 = 左右子树深度的最大值 + 1
The max depth = max(left subtree depth, right subtree depth) + 1

核心框架 / Core pattern:
- 空节点返回 0 / Empty node returns 0
- 递归求左右子树深度，取大的加1 / Recurse on left and right, take the max and add 1

---

## 代码 / Solution
```python
def maxDepth(self, root):
    if root is None:
        return 0
    left = self.maxDepth(root.left)
    right = self.maxDepth(root.right)
    return max(left, right) + 1
```

---

## 复杂度 / Complexity

- Time: O(n) — 每个节点访问一次 / visit every node once
- Space: O(h) — h 是树的高度，递归栈空间 / h is tree height, recursion stack

---

## 关键点 / Key Takeaway

这是二叉树递归的最基本模板：**先递归左右子树，再处理当前节点（后序位置）**。
几乎所有二叉树题都是这个框架的变体。

This is the most fundamental binary tree recursion template: **recurse left and right first, then handle current node (post-order)**. Almost every binary tree problem is a variation of this pattern.
