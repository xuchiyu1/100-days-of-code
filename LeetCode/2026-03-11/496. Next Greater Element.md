# LeetCode 496 – Next Greater Element I
## 单调栈（Monotonic Stack）| 中英文对照版

---

## 🧩 Problem Description | 题目描述

### English

Given two arrays `nums1` and `nums2` where `nums1` is a subset of `nums2`.

For each element in `nums1`, find the first greater element to its right in `nums2`.  
If none exists, return `-1`.

### 中文

给定两个数组 `nums1` 和 `nums2`，其中 `nums1` 是 `nums2` 的子集。

对于 `nums1` 中的每个元素，找到它在 `nums2` 中 **右侧第一个更大的数**。  
如果不存在，则返回 `-1`。

---

# 🎯 Core Idea | 核心思想

### English

This is a classic **Next Greater Element** problem.

We use:

- Reverse traversal (right → left)
- Monotonic decreasing stack
- Hash map for fast lookup

Time Complexity: O(n)

### 中文

这是典型的 **Next Greater Element（下一个更大元素）** 问题。

我们使用：

- 反向遍历（从右往左）
- 单调递减栈
- 哈希表加速查询

时间复杂度：O(n)

---

# 🔁 Why Reverse Traversal? | 为什么反向遍历？

### English

We are searching for the first greater element on the RIGHT.

When traversing from right to left:

- All elements to the right are already processed
- The stack contains valid future candidates
- We avoid repeated scanning

### 中文

我们要找的是：

> 右侧第一个更大的元素

当我们从右往左遍历时：

- 右侧元素已经处理过
- 栈中保存的是“未来可能成为答案”的候选
- 避免重复扫描

---

# 🧠 Monotonic Stack Rule | 单调栈规则

### English

The stack maintains decreasing order (by value).

If current value ≥ stack top value:

```python
while stack and nums2[i] >= nums2[stack[-1]]:
    stack.pop()
```

It means the stack top cannot be the next greater element.

### 中文

栈始终保持“值递减”。

如果当前值 ≥ 栈顶值：

```python
while stack and nums2[i] >= nums2[stack[-1]]:
    stack.pop()
```

说明：

- 栈顶不可能成为当前元素的“第一个更大”
- 直接淘汰

# 🔎 Step-by-Step Example | 逐步示例（重点）

Example:

```python
nums2 = [1, 3, 4, 2]
```

Goal: Compute next greater for each position.

目标：为每个位置计算 next greater。

---

## Initial State | 初始状态

```
stack = []
next_greater = [0,0,0,0]
```

---

## i = 3 (nums2[3] = 2)

### English

- Stack empty
- next_greater[3] = -1
- stack = [3]

### 中文

- 栈为空
- next_greater[3] = -1
- stack = [3]

---

## i = 2 (nums2[2] = 4)

### English

- 4 >= 2 → pop index 3
- stack becomes empty
- next_greater[2] = -1
- stack = [2]

### 中文

- 4 >= 2 → 弹出 3
- 栈为空
- next_greater[2] = -1
- stack = [2]

---

## i = 1 (nums2[1] = 3)

### English

- 3 >= 4 ❌ false → no pop
- stack not empty
- next_greater[1] = nums2[2] = 4
- stack = [2,1]

### 中文

- 3 >= 4 ❌ 不成立 → 不弹
- 栈不为空
- next_greater[1] = nums2[2] = 4
- stack = [2,1]

---

## i = 0 (nums2[0] = 1)

### English

- 1 >= 3 ❌ false → no pop
- stack not empty
- next_greater[0] = nums2[1] = 3
- stack = [2,1,0]

### 中文

- 1 >= 3 ❌ 不成立 → 不弹
- 栈不为空
- next_greater[0] = nums2[1] = 3
- stack = [2,1,0]

---

## ✅ Final Result | 最终结果

```
next_greater = [3, 4, -1, -1]
```

Meaning:

- 1 → 3
- 3 → 4
- 4 → -1
- 2 → -1

# 💻 Full Solution Code | 完整代码

```python
class Solution:
    def nextGreaterElement(self, nums1, nums2):

        result = [0] * len(nums1)

        # Step 1: Precompute next greater for nums2
        next_greater = [0] * len(nums2)
        stack = []

        for i in range(len(nums2) - 1, -1, -1):

            while stack and nums2[i] >= nums2[stack[-1]]:
                stack.pop()

            if not stack:
                next_greater[i] = -1
            else:
                next_greater[i] = nums2[stack[-1]]

            stack.append(i)

        # Step 2: Build mapping
        value_to_next = {}
        for i in range(len(nums2)):
            value_to_next[nums2[i]] = next_greater[i]

        # Step 3: Build result
        for i in range(len(nums1)):
            result[i] = value_to_next[nums1[i]]

        return result
```

# 🧠 Final Memory Rule | 最终记忆公式

```
Right-side next greater
= Reverse traversal + Monotonic decreasing stack

右侧第一个更大
= 反向遍历 + 单调递减栈
```
