# 695. Max Area of Island

**Difficulty:** Medium | **Topic:** Graph, DFS | **Date:** 2026-04-10

---

## 题目 / Problem

给定一个由0和1组成的网格，找出面积最大的岛屿面积（岛屿由相邻的1组成）。
Given a grid of 0s and 1s, find the maximum area of an island (connected 1s).

---

## 思路 / Approach

和200题完全一样的DFS淹岛法，唯一区别：
DFS返回淹掉的陆地格子数（面积），主函数取最大值。

Same DFS flood fill as LC 200, only difference:
DFS returns the area of the sunken island, main function takes the max.

---

## 与200题的对比 / Comparison with LC 200

        LC 200 岛屿数量:          LC 695 最大面积:
        res += 1                  res = max(res, self.dfs(...))
        dfs 返回 None             dfs 返回面积（int）
        数岛屿个数                 记录最大面积

---

## 代码 / Solution

    class Solution:
        def maxAreaOfIsland(self, grid):
            res = 0
            m, n = len(grid), len(grid[0])
            for i in range(m):
                for j in range(n):
                    if grid[i][j] == 1:
                        res = max(res, self.dfs(grid, i, j))
            return res

        def dfs(self, grid, i, j):
            m, n = len(grid), len(grid[0])
            if i < 0 or j < 0 or i >= m or j >= n:
                return 0
            if grid[i][j] == 0:
                return 0
            grid[i][j] = 0
            return self.dfs(grid, i + 1, j) \
                 + self.dfs(grid, i, j + 1) \
                 + self.dfs(grid, i - 1, j) \
                 + self.dfs(grid, i, j - 1) + 1

---

## 关键点 / Key Point

DFS返回值 = 四个方向的面积之和 + 1（当前格子本身）
DFS return = sum of areas in 4 directions + 1 (current cell itself)

---

## 复杂度 / Complexity

- Time: O(m×n)
- Space: O(m×n) — 递归栈 / recursion stack

---

## Key Takeaway

200和695是同一个模板的两种变体：
- 200：统计起点个数 → res += 1
- 695：统计淹掉的格子数 → dfs有返回值，返回面积

LC 200 and 695 are two variations of the same DFS template.
The only change is what you count and what DFS returns.
