# 102. Binary Tree Level Order Traversal

**Difficulty:** Medium | **Topic:** Binary Tree, BFS | **Date:** 2026-04-07

---

## 题目 / Problem

给定一个二叉树，返回其按层序遍历的节点值（每层一个列表）。
Given a binary tree, return the level order traversal of its nodes' values (one list per level).

---

## 思路 / Approach

BFS（广度优先搜索）+ 队列
BFS + Queue

关键点：进入每层前先记录 sz = len(q)
Key: record sz = len(q) before processing each level
这样即使队列里加入了下一层的节点，也能精确控制当前层处理几个。
This controls exactly how many nodes belong to the current level.

---

## 代码 / Solution

    from collections import deque

    class Solution:
        def levelOrder(self, root):
            res = []
            if root is None:
                return res
            q = deque()
            q.append(root)
            while q:
                sz = len(q)
                level = []
                for i in range(sz):
                    cur = q.popleft()
                    level.append(cur.val)
                    if cur.left is not None:
                        q.append(cur.left)
                    if cur.right is not None:
                        q.append(cur.right)
                res.append(level)
            return res

---

## 逐步图解 / Step-by-step

        树结构:
                1
               / \
              2   3
             / \
            4   5

        第1轮: sz=1, 处理[1], 加入2,3 → level=[1]
        第2轮: sz=2, 处理[2,3], 加入4,5 → level=[2,3]
        第3轮: sz=2, 处理[4,5] → level=[4,5]

        结果: [[1],[2,3],[4,5]]

---

## 复杂度 / Complexity

- Time: O(n) — 每个节点访问一次 / visit every node once
- Space: O(n) — 队列最多存一层节点 / queue holds at most one level

---

## Key Takeaway

sz = len(q) 是BFS分层的灵魂。
进入每层前快照队列长度，保证只处理当前层，不会混入下一层。

sz = len(q) is the key to level-by-level BFS.
Snapshot the queue size before each level to process exactly the current layer.
