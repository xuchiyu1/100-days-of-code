# 94. Binary Tree Inorder Traversal

**Difficulty:** Easy | **Topic:** Binary Tree, DFS | **Date:** 2026-04-05

---

## 题目 / Problem

给定一个二叉树的根节点，返回其中序遍历的节点值列表。
Given the root of a binary tree, return the inorder traversal of its nodes' values.

---

## 思路 / Approach

中序遍历 = 左 → 根 → 右
Inorder = Left → Root → Right

关键：`res.append` 放在中间位置
Key: put `res.append` in the middle position

---

## 代码 / Solution
```python
class Solution:
    def inorderTraversal(self, root):
        res = []
        def traverse(root):
            if root is None:
                return
            traverse(root.left)
            res.append(root.val)
            traverse(root.right)
        traverse(root)
        return res
```

---

## 三种遍历对比 / Three Traversals

| 遍历 | 顺序 | append位置 |
|------|------|-----------|
| 前序 Preorder | 根→左→右 | 最上面 |
| 中序 Inorder | 左→根→右 | 中间 |
| 后序 Postorder | 左→右→根 | 最下面 |

---

## 复杂度 / Complexity

- Time: O(n)
- Space: O(h) — 树的高度 / tree height

---

## Key Takeaway

三种遍历本质上是同一套框架，唯一区别是处理当前节点的时机。
All three traversals use the same framework — the only difference is when you process the current node.
