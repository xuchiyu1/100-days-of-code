# 198. House Robber

**Difficulty:** Medium | **Topic:** Dynamic Programming | **Date:** 2026-04-17

---

## 题目 / Problem

一排房子，每个房子有金额，不能抢相邻两个，求能抢到的最大金额。
A row of houses with money, cannot rob adjacent houses, find max amount.

---

## 思路 / Approach

记忆化递归（自顶向下DP）
Memoized recursion (top-down DP)

每个房子只有两个选择：
Each house has two choices:
- 抢：拿走nums[i]，下一个只能从i+2开始
- 不抢：直接去i+1

取两种选择的最大值。
Take the max of both choices.

---

## 状态转移方程 / State Transition

    dfs(i) = max(
        nums[i] + dfs(i+2),   # 抢当前，跳过下一个
        dfs(i+1)               # 不抢当前，去下一个
    )

---

## 代码 / Solution

    class Solution:
        def rob(self, nums):
            memo = {}

            def dfs(i):
                if i >= len(nums):    # 越界，没有房子可抢
                    return 0
                if i in memo:         # 算过了，直接返回
                    return memo[i]
                rob_this = nums[i] + dfs(i + 2)   # 抢当前
                skip_this = dfs(i + 1)             # 不抢当前
                memo[i] = max(rob_this, skip_this)
                return memo[i]

            return dfs(0)

---

## 为什么不需要dfs(i+3) / Why not dfs(i+3)

    隔三个的情况已经被自动覆盖：
    抢0 → skip_this → 不抢1 → skip_this → 不抢2 → 抢3
    递归自动穷举所有间隔，只需定义抢/不抢两个选择。

## 复杂度 / Complexity

- Time: O(n) — 每个房子只算一次
- Space: O(n) — memo + 递归栈

---

## Key Takeaway

DP两步走：
1. 定义子问题：dfs(i) = 从第i个房子开始能抢到的最大金额
2. 状态转移：每个房子抢或不抢，取最大值

DP two steps:
1. Define subproblem: dfs(i) = max amount starting from house i
2. Transition: rob or skip, take the max
