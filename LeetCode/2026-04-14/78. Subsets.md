# 78. Subsets

**Difficulty:** Medium | **Topic:** Backtracking | **Date:** 2026-04-14

---

## 题目 / Problem

给定一个不含重复数字的数组，返回其所有可能的子集。
Given an array of distinct integers, return all possible subsets.

---

## 思路 / Approach

回溯，但和46全排列不同：不用等track装满，每进入一层就记录一次。
Backtracking, but unlike LC 46: record result at every level, not just when full.

用start控制只往后选，防止重复。
Use `start` to only pick forward, preventing duplicates.

---

## 与46全排列的对比 / Comparison with LC 46

    46 全排列:                    78 子集:
    if len(track)==len(nums):     res.append(track[:])  ← 一进来就记录
        res.append(track[:])
    range(0, len(nums))           range(start, len(nums))  ← 不回头
    需要used数组                   不需要used，用start控制

---

## 代码 / Solution

    class Solution:
        def subsets(self, nums):
            res = []
            track = []

            def backtrack(start):
                res.append(track[:])       # 每层都记录
                for i in range(start, len(nums)):
                    track.append(nums[i])  # 做选择
                    backtrack(i + 1)       # 递归，下一层从i+1开始
                    track.pop()            # 撤销选择

            backtrack(0)
            return res

---

## 草稿纸图解 / Step-by-step (nums=[1,2,3])

    进入backtrack(0): 记录[]
      选1, track=[1]
      进入backtrack(1): 记录[1]
        选2, track=[1,2]
        进入backtrack(2): 记录[1,2]
          选3, track=[1,2,3]
          进入backtrack(3): 记录[1,2,3], range空返回
          pop, track=[1,2]
        pop, track=[1]
        选3, track=[1,3]
        进入backtrack(3): 记录[1,3], range空返回
        pop, track=[1]
      pop, track=[]
      选2, track=[2] ...以此类推

    最终: [], [1], [1,2], [1,2,3], [1,3], [2], [2,3], [3]
    共8个 = 2³

---

## 为什么每个append都有对应的pop / Why every append has a pop

    append(1)          ← 放入
      append(2)        ← 放入
      pop()            ← 移除2（马上）
    pop()              ← 移除1（递归回来后）

    每个append都对应一个pop，一一配对，没有例外。
    看起来"没移除"是因为递归还没回来，回来之后一定会pop。

## 复杂度 / Complexity

- Time: O(n × 2ⁿ) — 共2ⁿ个子集，每个长度最多n
- Space: O(n) — 递归栈深度

## Key Takeaway

回溯两题对比：
- 46全排列：装满才记录，用used防重复
- 78子集：每层都记录，用start防重复

Backtracking two patterns:
- LC 46: record when full, use `used` array
- LC 78: record every level, use `start` parameter
