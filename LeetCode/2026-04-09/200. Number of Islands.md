# 200. Number of Islands

**Difficulty:** Medium | **Topic:** Graph, DFS/BFS | **Date:** 2026-04-09

---

## 题目 / Problem

给定一个由'1'（陆地）和'0'（水）组成的网格，计算岛屿的数量。
Given a grid of '1's (land) and '0's (water), count the number of islands.

---

## 思路 / Approach

DFS 淹岛法：遍历整个网格，遇到'1'就计数+1，然后用DFS把整块岛屿全部淹掉变成'0'，防止重复计数。

DFS flood fill: traverse the grid, when we find a '1', increment count and use DFS to sink the entire island to '0'.

---

## 代码 / Solution

    class Solution:
        def numIslands(self, grid):
            res = 0
            m, n = len(grid), len(grid[0])
            for i in range(m):
                for j in range(n):
                    if grid[i][j] == '1':
                        res += 1
                        self.dfs(grid, i, j)
            return res

        def dfs(self, grid, i, j):
            m, n = len(grid), len(grid[0])
            if i < 0 or j < 0 or i >= m or j >= n:
                return
            if grid[i][j] == '0':
                return
            grid[i][j] = '0'
            self.dfs(grid, i + 1, j)
            self.dfs(grid, i, j + 1)
            self.dfs(grid, i - 1, j)
            self.dfs(grid, i, j - 1)

---

## 逐步图解 / Step-by-step

        初始网格:
        1 1 0
        1 0 0
        0 0 1

        发现(0,0)='1' → res=1 → DFS淹掉整块 → 变成:
        0 0 0
        0 0 0
        0 0 1

        发现(2,2)='1' → res=2 → DFS淹掉 → 结束

        结果: 2

## 边界条件 / Base Cases

- 越界：i < 0 or j < 0 or i >= m or j >= n
- 已是海水：grid[i][j] == '0'

两个条件缺一不可。

---

## 复杂度 / Complexity

- Time: O(m×n) — 每个格子最多访问一次 / each cell visited at most once
- Space: O(m×n) — 递归栈最坏情况 / worst case recursion stack

---

## Key Takeaway

DFS/BFS在网格里的应用 = 树的遍历换了个场景。
核心不变：找起点 → 标记访问 → 递归/入队邻居。

Grid DFS/BFS = tree traversal in a different setting.
Core stays the same: find start → mark visited → recurse/enqueue neighbors.
