# 300. Longest Increasing Subsequence

**Difficulty:** Medium | **Topic:** Dynamic Programming | **Date:** 2026-04-21

---

## 题目 / Problem

给定一个整数数组，找到最长严格递增子序列的长度。
Given an integer array, return the length of the longest strictly increasing subsequence.

---

## 思路 / Approach

自底向上DP（Bottom-up DP）

定义：`dp[i]` = 以`nums[i]`结尾的最长递增子序列长度
Definition: `dp[i]` = length of LIS ending at `nums[i]`

---

## 状态转移方程 / State Transition

    if nums[i] > nums[j]:
        dp[i] = max(dp[i], dp[j] + 1)

    含义：nums[i]可以接在nums[j]后面，长度就是dp[j]+1

---

## 代码 / Solution

    class Solution:
        def lengthOfLIS(self, nums):
            dp = [1] * len(nums)   # 初始化为1，每个数本身是长度1的子序列
            for i in range(len(nums)):
                for j in range(i):         # 只看i前面的数
                    if nums[i] > nums[j]:  # nums[i]可以接在nums[j]后面
                        dp[i] = max(dp[i], dp[j] + 1)
            return max(dp)                 # 最长子序列不一定以最后一个数结尾

---

## 图解 / Example (nums=[10,9,2,5,3,7,101,18])

    i=0: dp[0]=1  (10)
    i=1: dp[1]=1  (9，前面只有10>9，接不上)
    i=2: dp[2]=1  (2，前面都比2大，接不上)
    i=3: dp[3]=2  (2,5)
    i=4: dp[4]=2  (2,3)
    i=5: dp[5]=3  (2,3,7 或 2,5,7)
    i=6: dp[6]=4  (2,3,7,101)
    i=7: dp[7]=4  (2,3,7,18)

    max(dp) = 4

---

## 为什么不能直接return dp[-1] / Why not return dp[-1]

    nums = [3,1,2,0]
    dp   = [1,1,2,1]   ← 最后一个0接不了任何数，dp[-1]=1
    但答案是2（子序列[1,2]）
    所以必须遍历整个dp取最大值

---

## 为什么dp初始化为1 / Why init with 1

    每个数本身就是长度为1的子序列
    最少也是1，不可能是0

---

## 复杂度 / Complexity

- Time: O(n²) — 双层循环
- Space: O(n) — dp数组

---

## Key Takeaway

DP三要素：
1. 定义dp含义：dp[i] = 以nums[i]结尾的LIS长度
2. 状态转移：能接上就更新 dp[i] = max(dp[i], dp[j]+1)
3. 初始化：dp全为1，因为每个数本身是长度1的子序列

DP three elements: define → transition → initialize
