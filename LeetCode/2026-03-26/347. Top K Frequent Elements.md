# 347. Top K Frequent Elements

## 中文：前 K 个高频元素  
## English: Top K Frequent Elements

---

## 1. 题目 / Problem

给定一个整数数组 `nums` 和一个整数 `k`，返回出现频率最高的 `k` 个元素。

Given an integer array `nums` and an integer `k`, return the `k` most frequent elements.

---

## 2. 核心思路 / Core Idea

把问题拆成两步：

1. 用哈希表统计每个数字出现的次数
2. 用最小堆维护前 k 个高频元素

Split the problem into two steps:

1. Count frequency using a hash map
2. Maintain top k elements using a min heap

---

## 3. 为什么用堆 / Why Heap

堆适合做：

- 动态维护 Top K
- 快速删除“最小的那个”
- 保持结构始终是最优的 k 个候选

Heap is ideal for:

- Maintaining Top K dynamically
- Removing the smallest element efficiently
- Keeping only the best k candidates

---

## 4. 完整代码 / Full Code

```python
from collections import Counter
import heapq
from typing import List

class Solution:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:
        val_to_freqs = Counter(nums)
        pq = []

        for num, freq in val_to_freqs.items():
            heapq.heappush(pq, (freq, num))
            if len(pq) > k:
                heapq.heappop(pq)

        res = []
        while pq:
            res.append(heapq.heappop(pq)[1])

        return res[::-1]
```

## 5. 逐句解析 / Line-by-line Explanation

### Step 1：统计频率

```python
val_to_freqs = Counter(nums)
```

中文  
统计每个数字出现的次数。

示例：

```python
nums = [1,1,1,2,2,3]
```

得到：

```python
{1:3, 2:2, 3:1}
```

English  
Count frequency of each number.

---

### Step 2：初始化堆

```python
pq = []
```

中文  
创建一个最小堆（priority queue）。

English  
Initialize a min heap.

---

### Step 3：遍历频率表

```python
for num, freq in val_to_freqs.items():
```

中文  
遍历每个数字和它的频率。

English  
Iterate over number-frequency pairs.

---

### Step 4：入堆

```python
heapq.heappush(pq, (freq, num))
```

中文  
把 `(频率, 数字)` 放入堆中。

原因：

- heap 会按第一个元素排序（freq）
- 堆顶是最小频率

English  
Push `(frequency, number)` into heap.

Reason:

- heap compares first element
- smallest frequency stays on top

---

### Step 5：控制堆大小

```python
if len(pq) > k:
    heapq.heappop(pq)
```

中文  
如果堆超过 k，就删除最小频率的元素。

👉 核心作用：

始终保证堆里是“当前前 k 个高频元素”。

English  
If heap size exceeds k, remove smallest frequency.

👉 This ensures:

Heap always keeps top k frequent elements.

---

## ⭐ 核心逻辑总结

```python
for num, freq in val_to_freqs.items():
    heapq.heappush(pq, (freq, num))
    if len(pq) > k:
        heapq.heappop(pq)
```

中文  
不断加入元素，同时淘汰最小频率。

English  
Continuously add elements and remove the least frequent.

---

### Step 6：收集结果

```python
res = []
while pq:
    res.append(heapq.heappop(pq)[1])
```

中文  

- heappop 返回 `(freq, num)`
- `[1]` 取 num

English  

- heappop returns `(freq, num)`
- `[1]` extracts number

---

### Step 7：反转结果

```python
return res[::-1]
```

中文  
因为堆弹出顺序是从小到大，需要反转。

English  
Heap pops from small to large → reverse needed.

## 6. 示例流程 / Example Walkthrough

输入：

```python
nums = [1,1,1,2,2,3]
k = 2
```

---

### Step 1：统计频率

```python
{1:3, 2:2, 3:1}
```

---

### Step 2：构建堆

#### 加入 (3,1)

```python
[(3,1)]
```

---

#### 加入 (2,2)

```python
[(2,2), (3,1)]
```

---

#### 加入 (1,3)

```python
[(1,3), (3,1), (2,2)]
```

超过 k → 弹出：

```python
(1,3)
```

堆变为：

```python
[(2,2), (3,1)]
```

---

### Step 3：取出结果

```python
pop → (2,2) → 2
pop → (3,1) → 1
```

得到：

```python
[2,1]
```

反转：

```python
[1,2]
```

## 8. 常见错误 / Common Mistakes

### ❌ 错误1：缩进错误

```python

```

必须在 for 内部。

---

### ❌ 错误2：忘记反转

结果顺序会错。

---

### ❌ 错误3：理解错比较对象

比较的是：

```text
频率 freq
```

不是数值大小。

```python
ans = []
while heap:

return ans[::-1]
```
