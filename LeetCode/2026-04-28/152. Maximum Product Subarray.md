# 152. Maximum Product Subarray

**Difficulty:** Medium | **Topic:** Dynamic Programming | **Date:** 2026-04-28

---

## 题目 / Problem

给定一个整数数组，找到乘积最大的连续子数组，返回其最大乘积。
Given an integer array, find the contiguous subarray with the largest product and return it.

---

## 思路 / Approach

和53最大子数组和类似，但需要同时维护最大值和最小值。
Similar to LC 53, but must track both max and min products.

原因：负数×负数=正数，当前最小值乘以负数可能变成最大值。
Reason: negative × negative = positive, current min × negative may become max.

---

## 定义 / Definition

    dp1[i] = 以nums[i]结尾的子数组最小乘积
    dp2[i] = 以nums[i]结尾的子数组最大乘积

---

## 状态转移方程 / State Transition

    dp1[i] = min(dp1[i-1]×nums[i], dp2[i-1]×nums[i], nums[i])
    dp2[i] = max(dp1[i-1]×nums[i], dp2[i-1]×nums[i], nums[i])

    三个选项：
    - dp1[i-1]×nums[i]：最小值接上当前数
    - dp2[i-1]×nums[i]：最大值接上当前数
    - nums[i]：自己单独开始

---

## 代码 / Solution

    class Solution:
        def maxProduct(self, nums):
            n = len(nums)
            dp1 = [0] * n   # 最小乘积
            dp2 = [0] * n   # 最大乘积
            dp1[0] = dp2[0] = nums[0]
            for i in range(1, n):
                dp1[i] = min(dp1[i-1]*nums[i], dp2[i-1]*nums[i], nums[i])
                dp2[i] = max(dp1[i-1]*nums[i], dp2[i-1]*nums[i], nums[i])
            res = float('-inf')
            for i in range(n):
                res = max(res, dp2[i])
            return res

---

## 负数互换图解 / Negative Number Swap (nums=[2,3,-2])

    i=0: dp1=2,  dp2=2
    i=1: dp1=3,  dp2=6    （正数，不互换）
    i=2: nums=-2（负数，互换！）
      dp1 = min(3×-2, 6×-2, -2) = -12  ← 原来最大6×-2变成最小-12
      dp2 = max(3×-2, 6×-2, -2) = -2

    res = max(2, 6, -2) = 6

---

## 与53对比 / vs LC 53

    53 最大子数组和:          152 最大子数组积:
    只维护dp_i（最大和）      同时维护dp1（最小积）和dp2（最大积）
    负数直接抛弃重新开始       负数可能让最小变最大，不能抛弃

---

## 为什么要遍历dp2取最大 / Why scan dp2

    最大乘积不一定以最后一个数结尾
    nums=[2,3,-2] → 最大是6，在i=1，不是最后

---

## 复杂度 / Complexity

- Time: O(n) — 一次遍历
- Space: O(n) — dp1和dp2数组（可优化到O(1)）

---

## Key Takeaway

有负数的乘积问题必须同时维护最大和最小：
负数×最小（负）= 最大（正）
负数×最大（正）= 最小（负）

When negatives involved, always track both max and min:
negative × min(negative) = max(positive)
negative × max(positive) = min(negative)
