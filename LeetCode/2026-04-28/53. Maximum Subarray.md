# 53. Maximum Subarray

**Difficulty:** Medium | **Topic:** Dynamic Programming | **Date:** 2026-04-28

---

## 题目 / Problem

给定一个整数数组，找到和最大的连续子数组，返回其最大和。
Given an integer array, find the contiguous subarray with the largest sum and return its sum.

---

## 思路 / Approach

自底向上DP，空间优化版
Bottom-up DP with space optimization

定义：`dp_i` = 以`nums[i]`结尾的最大子数组和
Definition: `dp_i` = max subarray sum ending at `nums[i]`

---

## 状态转移方程 / State Transition

    dp_i = max(nums[i], nums[i] + dp_i)

    含义：
    - 选nums[i]：自己单独开始（前面是负贡献，拖累了）
    - 选nums[i] + dp_i：接在前面后面（前面是正贡献）

---

## 代码 / Solution

    class Solution:
        def maxSubArray(self, nums):
            n = len(nums)
            if n == 0:
                return 0
            dp_i = nums[0]   # base case：第一个数
            res = dp_i
            for i in range(1, n):
                dp_i = max(nums[i], nums[i] + dp_i)  # 接上还是单独开始
                res = max(res, dp_i)                  # 记录全局最大
            return res

---

## 图解 / Example (nums=[-2,1,-3,4,-1,2,1,-5,4])

    i=0: dp_i=-2, res=-2
    i=1: dp_i=max(1, 1+(-2))=1,    res=1
    i=2: dp_i=max(-3,-3+1)=-2,     res=1
    i=3: dp_i=max(4, 4+(-2))=4,    res=4
    i=4: dp_i=max(-1,-1+4)=3,      res=4
    i=5: dp_i=max(2, 2+3)=5,       res=5
    i=6: dp_i=max(1, 1+5)=6,       res=6
    i=7: dp_i=max(-5,-5+6)=1,      res=6
    i=8: dp_i=max(4, 4+1)=5,       res=6

    答案：6（子数组[4,-1,2,1]）

---

## 三个关键点 / Three Key Points

    1. dp_i初始化为nums[0]而不是0
       → 全负数时0会错误，如[-1,-2,-3]答案是-1不是0

    2. 选nums[i]还是nums[i]+dp_i
       → dp_i>0：接上去，正贡献
       → dp_i<0：单独开始，负贡献拖累了

    3. 需要res记录全局最大，不能直接return dp_i
       → 最大值可能在中间，最后的dp_i不一定最大

---

## 空间优化 / Space Optimization

    普通写法：dp数组 O(n)空间
    这里用dp_i单个变量替代数组，O(1)空间
    因为dp[i]只依赖dp[i-1]，不需要保存所有历史

---

## 复杂度 / Complexity

- Time: O(n) — 一次遍历
- Space: O(1) — 只用一个变量dp_i
