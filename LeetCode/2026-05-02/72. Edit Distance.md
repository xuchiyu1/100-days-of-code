# 72. Edit Distance

**Difficulty:** Hard | **Topic:** Dynamic Programming | **Date:** 2026-05-02

---

## 题目 / Problem

给定两个字符串s1和s2，计算将s1转换为s2所需的最少操作数。
操作包括：插入、删除、替换一个字符。
Given two strings, find the minimum number of operations to convert s1 to s2.
Operations: insert, delete, or replace a character.

---

## 定义 / Definition

    dp[i][j] = s1前i个字符变成s2前j个字符需要的最少操作数

---

## 状态转移方程 / State Transition

    if s1[i-1] == s2[j-1]:
        dp[i][j] = dp[i-1][j-1]      # 字符相同，不需要操作

    else:
        dp[i][j] = min(
            dp[i-1][j] + 1,           # 删除s1[i]
            dp[i][j-1] + 1,           # 插入s2[j]
            dp[i-1][j-1] + 1          # 替换当前字符
        )

---

## base case

    dp[i][0] = i   # s2为空，s1删i次
    dp[0][j] = j   # s1为空，插入j次

---

## 代码 / Solution

    class Solution:
        def minDistance(self, s1, s2):
            m, n = len(s1), len(s2)
            dp = [[0] * (n+1) for _ in range(m+1)]
            for i in range(1, m+1):
                dp[i][0] = i
            for j in range(1, n+1):
                dp[0][j] = j
            for i in range(1, m+1):
                for j in range(1, n+1):
                    if s1[i-1] == s2[j-1]:
                        dp[i][j] = dp[i-1][j-1]
                    else:
                        dp[i][j] = min(
                            dp[i-1][j] + 1,
                            dp[i][j-1] + 1,
                            dp[i-1][j-1] + 1
                        )
            return dp[m][n]

## 图解 / Example (s1="horse", s2="ros")

        ""  r  o  s
    ""   0  1  2  3
    h    1  1  2  3
    o    2  2  1  2
    r    3  2  2  2
    s    4  3  3  2
    e    5  4  4  3

    答案：dp[5][3] = 3
    操作：horse → rorse（替换h为r）→ rose（删除r）→ ros（删除e）

---

## 三种操作对应位置 / Three Operations

    dp[i-1][j-1] → 左上角 → 替换
    dp[i-1][j]   → 上方   → 删除s1当前字符
    dp[i][j-1]   → 左方   → 插入s2当前字符

---

## 复杂度 / Complexity

- Time: O(m×n) — 填满整个dp表
- Space: O(m×n) — 二维dp数组

---

## Key Takeaway

字符相同：不操作，继承左上角
字符不同：删/插/替三选一，取最小

记住三个方向：左上=替换，上=删除，左=插入
