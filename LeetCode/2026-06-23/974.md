# 974. Subarray Sums Divisible by K

**日期:** 2026-06-24
**类型:** 前缀和 + 哈希表(余数)
**难度:** Medium

## 题意
统计有多少个非空连续子数组,其和能被 `k` 整除。

## 核心思路
子数组和 `prefix[j] - prefix[i]` 能被 `k` 整除,**等价于两个前缀和对 k 的余数相同**:

```
(prefix[j] - prefix[i]) % k == 0
⟺ prefix[j] % k == prefix[i] % k
```

所以**不是去找余数为 0 的前缀和,而是统计"余数相同的前缀和对"**。每个余数桶(0, 1, ..., k-1)都同样地贡献配对。余数 0 唯一特别之处:它和初始的空前缀 `{0: 1}` 配对(代表从下标 0 开始的子数组)。

它是 LC 560(子数组和 == k)的兄弟题:560 配对"差为 k"的前缀和,974 配对"余数相同"的前缀和。

## 代码
```python
class Solution:
    def subarraysDivByK(self, nums, k):
        count = {0: 1}          # 余数 0 先出现 1 次(空前缀)
        prefix = 0
        res = 0
        for num in nums:
            prefix += num
            r = prefix % k
            if r in count:
                res += count[r]               # 和前面所有相同余数的前缀和配对
            count[r] = count.get(r, 0) + 1     # 登记当前余数
        return res
```

## 坑:负数取模
`nums` 可能为负。
- Python:`-5 % 3 == 1`(永远非负)→ 直接可用
- Java/C++:`-5 % 3 == -2`(可能负)→ 要写 `((x % k) + k) % k`

## 复杂度
- 时间 O(n),一次遍历
- 空间 O(min(n, k)),最多 k 个不同余数

## 一句话总结
前缀和 + 哈希表 = "配对",不是"找目标值"。560 按值差配对;974 按余数相同配对。
