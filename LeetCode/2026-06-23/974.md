# 974. Subarray Sums Divisible by K

**Date:** 2026-06-24
**Pattern:** Prefix Sum + Hash Map (remainders)
**Difficulty:** Medium

## Problem
Count the number of non-empty subarrays whose sum is divisible by `k`.

## Key Insight
A subarray sum `prefix[j] - prefix[i]` is divisible by `k`
**iff the two prefix sums have the same remainder mod k**:

So we don't look for prefix sums with remainder 0 — we **count pairs of
prefix sums that share the same remainder**. Every remainder bucket
(0, 1, ..., k-1) contributes pairs the same way. Remainder 0 is special
only because it pairs with the empty prefix `{0: 1}` (a subarray starting
from index 0).

This is the sibling of LC 560 (subarray sum == k): there we matched prefix
sums differing by `k`; here we match prefix sums with the **same remainder**.

## Solution
```python
class Solution:
    def subarraysDivByK(self, nums, k):
        count = {0: 1}          # remainder 0 seen once (empty prefix)
        prefix = 0
        res = 0
        for num in nums:
            prefix += num
            r = prefix % k
            if r in count:
                res += count[r]               # pairs with all earlier same-remainder prefixes
            count[r] = count.get(r, 0) + 1     # register current remainder
        return res
