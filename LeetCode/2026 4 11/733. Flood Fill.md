# 733. Flood Fill

**Difficulty:** Easy | **Topic:** Graph, DFS | **Date:** 2026-04-11

---

## 题目 / Problem

给定一个图像网格，从起点(sr,sc)开始，把所有与起点相连且颜色相同的格子换成新颜色color。
Given an image grid, starting from (sr,sc), change all connected pixels of the same color to the new color.

---

## 思路 / Approach

和200/695完全一样的DFS模板，改两个地方：
Same DFS template as LC 200/695, two changes:

- 停止条件：格子颜色 != 原始颜色（不是判断0）
- 操作：把格子换成新颜色（不是淹成0）

---

## 特殊情况 / Edge Case

新颜色 == 原颜色时必须提前返回，否则会无限递归。
If new color == original color, return early to avoid infinite recursion.

---

## 代码 / Solution

    class Solution:
        def floodFill(self, image, sr, sc, color):
            origColor = image[sr][sc]
            if origColor == color:
                return image
            self.dfs(image, sr, sc, origColor, color)
            return image

        def dfs(self, image, i, j, origColor, color):
            m, n = len(image), len(image[0])
            if i < 0 or j < 0 or i >= m or j >= n:
                return
            if image[i][j] != origColor:
                return
            image[i][j] = color
            self.dfs(image, i+1, j, origColor, color)
            self.dfs(image, i-1, j, origColor, color)
            self.dfs(image, i, j+1, origColor, color)
            self.dfs(image, i, j-1, origColor, color)

## 复杂度 / Complexity

- Time: O(m×n)
- Space: O(m×n) — 递归栈 / recursion stack

---

## Key Takeaway

网格DFS三题总结：
- 停止条件 = 越界 OR 不满足条件
- 操作 = 标记当前格子（防止重复访问）
- 递归四个方向

Grid DFS pattern: check boundary + condition → mark current → recurse 4 directions.
